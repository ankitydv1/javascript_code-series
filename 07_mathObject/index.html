Maths in javascript

  • addition operator: +
  • Substraction operator: -
  • Multiplication operator: *
  • Division operator: /
  • Remainder/modulus operator: %
  • Exponentiation operator: **
  • Math.sqrt(x): Square root of any number
  • Math.pow(2,6): Exponential power of any number
  • Math.floor(x):The math.floor() method rounds a number down to the nearest integer, if necessary, and returns the result.
  • Math.ceil(x) : The math.ceil() method rounds a number UP to the nearest integer, if necessary, and returns the result.
  • Math.round(x) : It is used to obtain round-off of any decimal number
  • Math.random() : It is used to obtain any random number
